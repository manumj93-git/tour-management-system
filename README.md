# 🧳 Tour Management System

## 📌 Overview

**Blazor Web Application** for managing tours, bookings, and users using clean architecture principles.

## 🚀 Features

* 🔐 User Authentication & Authorization
* 🧾 Tour Management (Create, Read, Update, Delete)
* 📦 Booking System
* 👨‍💼 Role-Based Access (Admin/User)
* 🏗️ Clean Architecture (Repository + Service Pattern)

## 🛠️ Tech Stack

* ASP.NET Core
* Blazor
* Entity Framework Core
* SQL Server
* AutoMapper

## 📂 Architecture

* Data (DbContext)
* Model
* DTO
* Helper (DTO Mapping)
* Interface
* Service
* Repository
* Extension
